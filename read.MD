#Proceso para ejecutar el software
Para poder ejecutar el proyecto primero se tiene que ejecutar el backend en la terminal de visual studio (ctrl + ñ para abrirla) de la siguiente manera:

- cd backend
- node server.js

Luego veras el puerto por donde se esta corriendo el Backend y una confirmacion de que la base de datos esta activada

Despues de ejecutar el backend, luego podras ejecutar el frontend  (tienes que abrir una nueva terminal de visual studio code (ctrl + shift + ñ)) de la siguiente manera :

- Abre una nueva terminal
- cd .. (solo aplica si sigues en la carpeta del backend)
- cd frontend
- npx http-server -p 5501

Luego de que el frontend y el backend esten corriendo en ambas terminales podras ver el menu del login abriendo el navegador y buscando lo siguiente:

- localhost:5501/login

Nota: Esta es una recomendacion para ejecutar el frontend pero se puede ejecutar tambien usando extensiones como open live server.
