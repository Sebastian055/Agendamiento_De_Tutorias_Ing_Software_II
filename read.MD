#Proceso para ejecutar el software
Para poder ejecutarlo primero se tiene que ejecutar el backend de la siguiente manera:

- cd backend
- node server.js

Luego veras el puerto por donde se esta corriendo el Backend y una confirmacion de que la base de datos esta activada

Despues de ejecutar el backend, luego podras ejecutar el frontend de la siguiente manera:

- cd .. (solo aplica si sigues en la carpeta del backend)
- cd frontend
- npx http-server -p 5501

Nota: Esta es una recomendacion para ejecutar el frontend pero se puede ejecutar tambien usando extensiones como open open live server.
